# haha

Terminal CLI that prints a trollface and can animate it as rain.

## Install

Install with Go:

```sh
go install github.com/Puker228/haha@latest
```

## Usage

Print one trollface:

```sh
go run .
```

Run falling trollface rain:

```sh
go run . rain
```

In rain mode, press `q` or `ctrl+c` to quit.
